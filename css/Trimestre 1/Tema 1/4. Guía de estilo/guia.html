  • Colores o paleta cromática

    Fuente

  • Tipografía

    Para textos neutrales Utilizaria la fuente de letra Segoe UI, la misma utilizada por plantilla Indigo de w3schools.

    Para textos de una temática concreta, buscaría fuentes personalizadas que casasen con la misma.

    Por ejemplo, si hablase de baloncesto, buscaría una fuente de letra similar a la utilizada por ligas como NBA o ACB

  • Botones y elementos de navegación

    Apostaría por los siguientes elementos:

    • Tener un banner en la cabecera un menú de navegación debajo
    • Una barra lateral a la izquierda con opciones
    • El contenido en bloque en el centro
    • Intentaría que los botones sean imágenes apagadas que se iluminen con efecto hover

  • Uso de logotipos e imágenes

    El banner principal sería una imagen diseñada para la web

    Para artículos descriptivos, las imágenes acompañarán a cada párrafo por el lateral, para amenizar la lectura

    Para mostrar artículos de venta, las imágenes acompañarían al enlace de compra de cada artículo, así como a su descripción mediante una galería.

  • Patrón de lectura

    El contenido principal de la página iría en un bloque centrado y se leería de arriba a abajo. Se dejaría un menú lateral a la izquierda y a la derecha un espacio para publicidad. También se pondría un espacio para publicidad antes del Footer
